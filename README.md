# weather-by-city
Get the weather forecast by city  

# Description
Visualize the temperature forecast for the next 24 hours in the specified city  

# Installation
git clone https://github.com/igorkond/weather-by-city  
cd weather-by-city  
npm install  

# Run server
node dist/index.js  

# Run client
http://127.0.0.1:8080/
